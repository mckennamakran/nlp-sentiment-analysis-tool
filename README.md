# 📊 NLP Sentiment Analysis Tool

## Overview
A simple **Natural Language Processing (NLP)** project built using Python and TextBlob to analyze the sentiment of user input in real time.

**Author:** McKenna Makran  
**Language:** Python  

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![TextBlob](https://img.shields.io/badge/TextBlob-NLP-lightgrey?style=for-the-badge)

## 🚀 Features
- Analyze text sentiment as:
  - Extremely Positive
  - Positive
  - Neutral
  - Negative
  - Extremely Negative
- Displays:
  - Polarity Score (emotional tone)
  - Subjectivity Score (fact vs opinion)
- Continuous input loop (until user exits)

## 🧠 How It Works
This project uses the **TextBlob** library to process text and extract sentiment information:

- Polarity:
  - Range: -1 to +1
  - Measures how positive or negative the text is

- Subjectivity:
  - Range: 0 to 1
  - Measures whether the text is factual (0) or opinion-based (1)
